General

This section documents the common API components of nvFuser.

Statement

class nvfuser_direct.Statement

Statement represents a base class for all statements in the nvFuser IR. It provides methods for:

  • Accessing statement properties

  • Managing statement dependencies

  • Querying statement type and attributes

class nvfuser_direct.FusionDefinition[source]

A class for defining and executing fused operations in nvFuser. This class provides a context manager interface for defining fused operations, managing inputs/outputs, and executing the fusion with PyTorch tensors. .. rubric:: Examples

>>> with FusionDefinition() as fd:
...     t0 = fd.define_tensor()
...     t1 = fd.ops.relu(t0)
...     fd.add_output(t1)
>>> outputs = fd.execute([input_tensor])

The DirectFusionDefinition class provides a low-level interface for defining nvFuser operations. It exposes two main components:

  • ops: Contains the operators for defining fusion operations
